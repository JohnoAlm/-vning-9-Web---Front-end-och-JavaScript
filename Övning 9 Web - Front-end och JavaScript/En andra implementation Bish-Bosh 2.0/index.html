What is Bish-Bosh?

Bish-Bosh is a game about numbers. First you set the value where the loop should end. This "loop" is a sort of counter you could say that displays all the numbers from
1 to the amount you set. For example if i set the number 100 the "loop" or counter will diplay all the numbers from 1 to 100, 1,2,3,4,5 and so on... After you have entered your
loop-value you set your Bish number. This can be any number from within 0-100, at least on this webpage. The Bish number is a sort of
division number and when it is in the loop it will check whether it is dividable by the numbers in that loop. If so then the number will be replaced by Bish!
